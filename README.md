# 🏆 SkillSangh

**SkillSangh** is a specialized platform designed for orchestrating technical events, hackathons, and workshops. It serves as a bridge between organizers and tech enthusiasts, streamlining registration, team management, and event execution.

---

## 🚀 Overview

Organizing a tech event involves many moving parts. SkillSangh simplifies this by providing a centralized hub for:
* **Hackathon Management:** Smooth registration flows and team formation.
* **Event Scheduling:** Real-time updates for workshops and seminars.
* **Participant Engagement:** Integrated tools to keep the community informed.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Backend:** Node.js (Express.js)
* **Data Handling:** JSON (JavaScript Object Notation)
* **Version Control:** Git & GitHub

## ⚙️ Getting Started
To get this project running locally, follow these steps:

**1. Clone the repository**
bash
git clone [https://github.com/your-username/SkillSangh.git](https://github.com/your-username/SkillSangh.git)

**2. Install Dependencies**
Make sure you have Node.js installed, then run:
bash
npm install

**3. Start the Server**
bash
node server.js

**4. Access the Web App**
Open your browser and go to http://localhost:3000 (or the port specified in your server.js).


---

## 💡 Usage
Organizers can utilize the backend via server.js to serve event data stored in JSON format. The frontend script.js dynamically fetches this data to update the UI.
